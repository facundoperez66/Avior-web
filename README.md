# Avior — Estudio digital

Sitio web del estudio Avior. HTML + CSS + JavaScript vanilla, desplegado en Vercel.

## Estructura

- `index.html` — Landing principal con intro animada
- `proyectos.html` — Página de portfolio
- `css/`, `js/`, `assets/` — Recursos estáticos
- `api/` — Función serverless del formulario de contacto (próximamente)

## Desarrollo local

Cualquier servidor estático sirve:

```bash
npx serve .
```

Abrir http://localhost:3000

## Deploy

Push a `main` → Vercel deploya automáticamente.
